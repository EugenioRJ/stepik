Дано целое число 1 ≤ n ≤ 40, необходимо вычислить n-е число Фибоначчи (напомним, что F0=0, F1=1, и Fn = Fn−1 + Fn−2 при n ≥ 2)

**Sample Input**:

3

**Sample Output**:

2

# Решение / Solution

```
def fib(n):
    assert n >= 0
    f0, f1 = 0, 1
    for i in range(n - 1):
        f0, f1 = f1, f0 + f1
    return f1


def main():
    n = int(input())
    print(fib(n))


if __name__ == "__main__":
    main()
```
